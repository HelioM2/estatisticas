1 ->  Verificar se o Git está instalado, Abra o terminal no VS Code e digite:

git --version
2 -> Navegar até o diretório do seu projeto no VS Code, Se o projeto já existe no seu computador, abra o terminal do VS Code e vá até a pasta.

3->Inicializar o repositório Git

git init

4-> Adicionar a origem do repositório GitHub, No GitHub, copie o link do repositório (HTTPS ou SSH) e execute o seguinte comando:

git remote add origin https://github.com/HelioM2/estatisticas.git

  - Verifique se a conexão foi estabelecida corretamente:

 git remote -v

5->  Adicionar e enviar arquivos para o GitHub, Agora, adicione os arquivos do seu projeto ao repositório e faça o primeiro commit:
git add .
git commit -m "Primeiro commit"
git branch -M main
git push -u origin main
